# About Streamsight

## Origin and Purpose

Streamsight is an open-source Python toolkit developed as an Final Year Project at
Nanyang Technological University (NTU), Singapore. The project was undertaken by
under the supervision of Professor Sun Aixin. This project was later
imporved upon to add UI to the package to expose the library's functionalities to
non-programmers.
